# Banner Generator Skill

## What this skill does
This skill generates on-brand Jio banner artworks across multiple
sizes from simple text inputs. Give it a headline, a campaign 
theme, and a size — it gives back 3-4 ready-to-use banner options.

## Who uses this
Anyone at Jio who needs campaign banners fast — without a designer.

## What the user needs to provide
- Headline text (e.g. "Unlimited Entertainment")
- Sub-copy (e.g. "Starting at ₹399/month")
- CTA text (e.g. "Get Jio Fiber Now")
- Campaign theme (e.g. product launch / festive / recharge promo)
- Mood/tone (e.g. energetic / calm / premium)
- Target platform (e.g. Instagram / Facebook / Website)
- Banner size(s) (e.g. 1080x1080 or "all social formats")

## What the tool outputs
- 3 to 4 visually distinct banner variations
- All following Jio brand guidelines (see /design-system/jio-tokens.md)
- Exportable as PNG, SVG, and PDF in a ZIP file

## Design system reference
All colours, fonts, spacing, and logo rules live in:
/design-system/jio-tokens.md
